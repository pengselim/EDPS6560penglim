Grid Elements

A ការ​ធ្វើ​ដូច្នេះ​គឺ​ដើម្បី​កុំ​ឱ្យ​មានការ​ចោទប្រកាន់ ឬ​ការ​យល់​ច្រឡំ​ពី​ក្រុម​ប្រទេស​លោក​ខាងលិច​ថា កម្ពុជា​ផ្អៀង​ទៅ​រក​ចិន​កាន់តែ​ខ្លាំង​ឡើងៗ ខណៈ​ប្រទេស​នេះ ចាប់ផ្ដើម​ធ្វើ​សមយុទ្ធ​យោធា «​នាគ​មាស​» ជាមួយ​ចិន ក្នុង​ទ្រង់ទ្រាយ​ធំ​ជាង​លើក​មុនៗ រយៈពេល ២៨ ថ្ងៃ ចាប់ពី​ថ្ងៃទី២៨ កុម្ភៈ​នេះ ដល់​ថ្ងៃ​ទី២៧ មីនា display property set to grid or inline-grid.

Direct child element(s) of the grid container automatically becomes grid items.
